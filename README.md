
This project is a dedicated e-commerce platform tailored for a professional tattoo supply shop based in Padua, Italy.

Currently in active development, the platform is designed to bridge the gap between high-end artistry and accessibility. Our mission is to curate an extensive catalog of premium-grade tattoo equipment—ranging from precision machines to specialized pigments—offering them at highly competitive prices.

## 🚀 Features

* 🛍️ Product catalog
* 🛒 Shopping cart
* 💳 Secure payments with Stripe
* 🗄️ MongoDB database
* 📱 Responsive UI

---

## 🛠 Tech Stack

* **Node.js**
* **Express**
* **MongoDB**
* **Stripe API**
* **JavaScript / HTML / CSS**

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YCEslow999/tattoo-supply-shop-padova.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create a `.env` file

```env
STRIPE_SECRET_KEY=your_key
MONGO_URI=your_mongo_uri
```

### 4. Run the project

```bash
npm run dev
```
